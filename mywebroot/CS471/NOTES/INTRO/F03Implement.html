Syntax vs Semantics

  •  Syntax of a programming language is the form of its expression, statements and program units.
    • This is usually described by a CFG grammar
    • Lexical analysis - usually described by RE
  • Semantics of a programming language is the meaning of the expressions, statements and program units.
  • Phases of Compilation

Names and Attributes

  • Names (identifiers)  are a fundamental abstraction in a programming language.
  • Attributes are properties associated with a name.
  • Binding is the association of a name and an attribute
    • Binding time when this association takes place
      • tension between speed and flexibility.
    • Static vs Dynamic properties
