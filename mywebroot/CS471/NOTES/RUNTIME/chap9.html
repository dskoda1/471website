C++ Example:
  • public, protected and private
  • All members (fields and methods) of a class are private by default (no code outside the class can reference them), whereas fields of a struct are public, meaning that anyone can use them.
    • C++:: derived class can restrict the visiability of of the base class
    • Java, C# can NOT.  Use composition to get the simular effect.
  • class circle : public shape { ...
    anybody can convert (assign) a circle* into a shape*
  • class circle : protected shape { ...
    only members and friends of circle or its derived classes can convert (assign) a circle* into a shape*
  • class circle : private shape { ...
    only members and friends of circle can convert (assign) a circle* into a shape*
A few C++ features:
  • Constructors:  every object has to be initialized before it is used.
    •  C++ uses the derived class header to specify the base class constructor arguments:

      foo::foo (args0) : member1 (args1), member2 (args2) { ...
          
      args1 and args2 need to be specified in terms of args0

    • Java: construct code may have an initial line with a call to super

  • initialization v. assignment
    foo::operator=(&foo) vs. foo::foo(foo&) (copy constructor)
    foo x;       // calls foo::foo ()
    foo f = x;   // calls foo::foo(foo&)
    foo x, f;    // calls no-argument constructor
    f = x;       // calls operator=

    foo a, c, d;     // calls foo::foo () 3 times
    bar b;           // calls bar::bar()
    foo y = a        //    calls foo::foo(foo&)
    foo z = b        // calls foo::foo(bar&)
    ...
    c = a;           // calls foo::operator=(foo&)
    d = b;           // calls foo::operator=(bar&)
