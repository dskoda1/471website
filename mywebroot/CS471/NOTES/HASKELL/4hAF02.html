Dynamic Dispatching: 
Which code is used? What is the Static/Dynamic Type?
class Animal { 
    void move(Animal x ) {
       code 1
    }
}		 class Bird extends Animal {
Animal ted;    // static type Animal
. . .
   void move(Animal x ) { code 2 }
   void fly( ){
ted = new Animal( ); //dyn type Animal
this.move(ted);   // uses code 2
ted.move(ted);   // uses code 1
ted = new Bird( );// dyn type Bird
this.move(ted);   // uses code 2
ted.move(ted);    // uses code 2
    }
}
