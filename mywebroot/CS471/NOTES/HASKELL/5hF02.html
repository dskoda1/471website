Simple functions you can find in Prelude

  • Testing for an empty list:
    • > null :: [a] -> Bool
    > null []    = True
    > null (_:_) = False
  • Testing for even and odd numbers:
    • > even, odd :: a -> Bool
    > even n = n `rem` 2 == 0
    > odd = not . even

not, ord and chr

> not :: Bool -> Bool
> not True  = False
> not False = True

> ord  :: Char -> Int
> ord   = fromEnum

> chr  :: Int -> Char
> chr   = toEnum

zip example from Prelude

(Alternate definition to the one in Prelude)
> zip :: [t] -> [u] -> [(t,u)]
> zip [] x = []
> zip x [] = []
> zip (a:x) (b:y)=(a,b):zip x y
  • EXAMPLE
    • ...> zip [1,2,3,4] ['a','b','c']
    [(1,'a'),(2,'b'),(3,'c')]

    zip3 example

      > zip3 :: [t]->[u]->[v] -> [(t,u,v)]
    • definition is similar but longer
      • ...> zip3 [1,2,3,4] ['a','b','c'] ["here","there”]
    • returns
      • [(1,'a',"here"),(2,'b',"there")]

    k-th element of a list, starting from 0

      >  (!!):: [b] -> Int -> b
      >  (x:_)  !! 0         = x
      >  (_:xs) !! n | n > 0 = xs !! (n-1)
      >  (_:_)  !! _         = error "Prelude.!!: negative index"
      >  []     !! _         = error "Prelude.!!: index too large"
    • EXAMPLE
      • ...> [1,2,3,4,5,6]!!3
      returns 4

    zipWith

      zipWith :: (t -> u -> v) -> [t] -> [u] -> [v]
    • applies function argument to corresponding elements of the two input lists to get a list of results
    • EXAMPLES
      • ...> zipWith (+) [1,2,3,4] [5,3,1]
    • returns
      • [6,5,4]

    zipWith example

      ...> zipWith (++) [[],[1],[1,2]] [[1,2,3],[2,3]]
    • returns
      • [[1,2,3],[1,2,3]]

    Preludes definition of zip

    > zip :: [t] -> [u] -> [(t,u)]
    > zip =  zipWith (,)

    Constructors can be considered special functions.

    > data People3 = Person3 String Int deriving Show

    > build :: [String] -> [Int] -> [People3]
    > build strs nums  = zipWith Person3 strs nums

    ..> zipWith Person3 ["ab", "joe", "jack"] [30, 43, 21]
    [Person3 "ab" 30,Person3 "joe" 43,Person3 "jack" 21]
