CS471 Homework Instructions

The purpose of the written assignments are to help you understand and apply programming language concepts to practical (common) languages.  Therefore to pass this course you must have an average of 65% on the written assignments.  You are expected to turn in homework every week even if you can not complete every question.  You should document any problems with your code not meeting specifications.

This semester all homework assignments will turned into the digital box in BLACKBOARD:

Time deadlines and ecopy requirements: 
  1. All code will be plain text file.  All written assignments will be either plain text or pdf files.  If you submit any other format you will be asked to resubmit. Make sure your name and bmail appear at the top of the file.
    • Written assignments will be done in ONE file. If there are specific questions related to coding code should be submitted in a separate file in the tar. It is expected that all code can be compiled and run.
    • Prolog and Haskell assignments should be in ONE .pl or .lhs file. If there are written questions for the assignments, they will be answered in COMMENTS included in the file.

      gzip acanino1_lab1.tar
