  • Syntax and semantics
    • Syntax of a programming language is the form of its expression, statements and program units.
      • This is usually described by a grammar
    • Semantics of a programming language is the meaning of the expressions, statements and program units.
      • Variables
        • corresponds to a memory region that holds values which are manipulated by the program
        • referred to by name
        • how variables are name is specified by the syntactic rules.
        • important semantic issues
          • scope
          • type
          • lifetime
      • Values and references
        • x = y
        • x = &y
        • 3 = y
        • y = 3
      • Expressions
        • x = y = z - w
        • order of operations
        • kinds of values can the expression return
    • Program organization
    • Many issues surround the idea of data
      • where, when, grouping, exporting, initialization...
    • Control structures.
