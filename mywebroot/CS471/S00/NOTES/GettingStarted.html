Installation

  • For Windows 95/98/NT, download Hugs 98 from:  http://www.cse.ogi.edu/PacSoft/projects/Hugs/downloading.htm (search on that page for the InstallShield version of Hugs98).
  • Local version of InstallShield for Nov 99
  • From the same page you can obtain the same program for Unix platforms and Macintosh.
  • When you download and run the Hugs98install.exe for Microsoft Windows, you first get a window asking you to extract the software (see graphic: haskellinstall1.jpg)
  • During the installation, please ensure that Hugs for Windows (winhugs) is installed. We would like you to use winhugs since it is much easier at the beginning (see graphic:  haskellinstall2.jpg)
  • We assume you use the default installation location: C:\hugs98
  • Life is easier if you move shortcuts to hugs.exe and winhugs.exe to the desktop, where you can use Properties to change the icons. There are icons in the directory C:\hugs98\icons
  • If you use the Properties of the winhugs.exe program you should be able to find the icon 
  • For example, you could choose the icon  for hugs.exe.

    • References for getting started:

    • Thompson chapter 2 (pages 19-31) and section 3.7 (pages 46-51)
    • The Help files that come with winhugs.exe (see below for how to find them).

    Starting hugs:

    • To use Haskell on bingsuns (presumably also on other Unix systems), type "xhugs"
    • On Windows 98, use the icon on the desktop or select winhugs.exe from the directory C:\hugs98

    Effect of selecting hugs.exe

    • You get a black workspace
    • There is no menu bar and no icon bar. Type ":q" to exit
    • See graphic: hugsScreen.jpg

    Effect of selecting winhugs.exe

    • White workspace with menu bar and icon bar. Exit by pressing the lowest icon or ":q"
    • See graphic: winHugsScreen.jpg

    Leaving hugs

    • When you teach someone to drive, always tell them how to stop before telling them how to go!
    • To exit HUGS, type ":q"

    Setting up the environment in winhugs

    • Setting the path to your own Haskell scripts in the shortcut:
      • C:\hugs98\winhugs.exe -Pc:\hugs98\lib;<path to your scripts> (see graphic: winhugsprops.jpg)
    • Alternatively, you can set the path and the editor of your choice in the Options:
      • Click on Options, click on Set (see graphic: defaultoptions.jpg) and add the path to your scripts and your editor of choice (see graphic: changedoptions.jpg)
