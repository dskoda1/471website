    DETAILS for problem 5:

    Horner's rule is an algorithm to evaluate a polynomial requires only a total of only n-1 multiplications of x. 

    If p(x) = x^3 + 5*x^2 + 6*x +7  then Horner's algorithm will calculate p(x) in this order:

((x + 5)*x +6 )*x +7

    Therefore there are only a total of 3 multiplications verse 2 +2+1+0 multiplication if you do it the "normal way".


    We can represent the coefficients of a polynomal anx n + an-1xn-1 +...+ a0 using a list = [  a0 ,..., a n-1 , a n].
    This algorithm is related to synthetic division you learned in high school.

       
    Below is an implementation of Horner's algorithm using Java:
      public class Example {
        public static int horner (int[] a, int n, int x) {
          int result = a[n];
          for (int i = n - 1; i>=0; --i) {
            result = result * x + a[i]
          }
          return result;
        }
      }
