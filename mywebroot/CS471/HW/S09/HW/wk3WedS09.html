    3. ?- italian(F).

    ?- emacs(likes).

       likes(martha, Food):-italian(Food).  
    (NO space between likes and "(")
    save the current buffer
    In the shell window reload the modified file.  You can find this command on the Pull down File menu.
    ?- likes(Name,Food).

After 'likes.pl' is loaded, try using help and emacs/edit etc:
At the prompt type:

To edit your file type:
