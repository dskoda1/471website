    Problem 10:
    The value of ex can be computed using the series expansion
     

         ex  = 1 + x + (x2 / 2!) + ( x3 / 3!) + ... + ( xn / n!) + ...
    In calculating ex we will sum all terms until ( xn / n!) < eps, where eps is specified in our problem to be 0.000001.
     

    We can use infinite lists to generate each term of the series and use takeWhile test to terminate the generation of the terms used in the series expansion. See  Fibonacci and  Newton method for calculating the square root for help in generating and working with infinite data structures.
