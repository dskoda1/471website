  1. The BookCompany program creates an inventory by adding "Book"s to the inventory. The program can add "Book"s to the inventory, sort the inventory and printout the inventory.  It is implemented in three paradigms using Haskell,Java and C. All three implementations use the generic sort method supplied in their respective library.  The sorting order of the "Book"s is based on the ascending order of year, followed by the ascending lexical order of the book title.
      1. The Haskell version, BookCompany.lhs  is implemented literal style. It uses the sort definition found in List.hs. In order to use the sort the type Book must be in the class Ord. Therefore the user must declare Book to be in class Ord as well as implement  the expected methods.
      2. The Java implementation, BookCompany.java, uses the static method  sort found in the java.util.Collections class .  Sort uses a modified mergesort to sort a list of mutually comparable elements.  The user must supply an ordering of "Book" to the sorting method.  This is done by implementing the interface Comparator for the elements type in the list and passing the an object of the class that implements it.  This version uses a nested class and an innerclass.
      3. The "C" implementation,  bookCompany.c use the stdlib's qsort to sort the inventory.  The user must supply an ordering function to the qsort for the data being sorted.  This is done by implement a function for ordering following the same rules as compare in Comparator in Java and then passing the function as one of the parameter.
    1. Java expects when you implement compare that your function imposes a total ordering on some collection of objects.   What is a total ordering?  Can you give an example of something that is not a total ordering?

Expand the book type (in above) to include fields for an authors first and last name.  The sorting order should now be based on the ascending lexical order of author's last name and then the ascending lexical order of book title.  You will need to update the testing data to reflect the addition information included.  You should make sure that thereis a way to printout all the information about the book.
