    DETAILS for problem 4:

    Horner's rule is an algorithm to evaluate a polynomial without requiring the computation of arbitrary powers of x.
    We can represent the coefficients of a polynomal anxn + an-1xn-1 +...+ a0 using a list = [  a0 ,..., an-1 , an].
    This algorithm is related to synthetic division you learned in high school.

       
    Below is an implementation of Horner's algorithm using Java:
      public class Example {
        public static int horner (int[] a, int n, int x) {
          int result = a[n];
          for (int i = n - 1; i>=0; --i) {
            result = result * x + a[i]
          }
          return result;
        }
      }
