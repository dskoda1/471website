Solution

(From Abelson and Sussman [1985]) It is also possible to express the integer as lambda abstractions:
zero = l f  . l x . x
one =  lf.  l x ( f x)
two =  l f. l x ( f ( f x ))
. . . .

These are called Church numbers after Alonzo Church.

a) Given the following definition of the sucessor function:

successor = l n. l f. l x ( f (( n f ) x ) ).
    show that (successor zero) = one  and (successor one) = two
 

one = "successor zero"
one = l n. l f. l x ( f (( n f ) x ) ) (l f  . l x . x)
      => l f. l x ( f ((  l f  . l x . x) f ) x))
      =>  l f. l x ( f (( l x . x) x))
      => l f. l x ( f x)

two = "successor one"
one = l n. l f. l x ( f (( n f ) x ) ) (l f  . l x .(f x))
      => l f. l x ( f ((  (l f  . l x . (f x)) f ) x))
      =>  l f. l x ( f (( l x .(f x)) x))
      => l f. l x (f ( f x) )
