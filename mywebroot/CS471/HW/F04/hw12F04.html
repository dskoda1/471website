  2. Prove formally that map can be defined nonrecursively using foldr. Specifically, prove
    that for all finite lists xs,
    map f xs = foldr ((:).f) [] xs
    The definitions, for reference:
    > map f [] = []
    > map f (a:as) = f a : map f as
    > foldr f k [] = k
    > foldr f k (b:bs) = f b (foldr f k bs)
    > (g.h) z = g (h z)
  3. Determine the types of the following functions:
    1. f0 m n [] = n
    f0 m n (x:xs) = f0 m (m n x) xs
    1. f2 g [] = []
    f2 g (b:c)
         | g b = b : f2 g c
         | otherwise = []
