  4. Defined a predicate "simplify/3", simplify(Var, Value, Lst), that succeeds if the last argument, Lst, is list of the form Var:Value. The first argument, Var, is an atom representing a variable and Value represents the value of that Var. This requires only one clause. Hint: you can use member & atom built in predicates. Note atom is an extra check but not required. e.g.

      5. ?- simplify(b,Value,[a:100,b:(-5)]).
      Value = -5
      ?- simplify(b,Value,[a : 1,b : 5, c : 10]).
      Value = 5 .
