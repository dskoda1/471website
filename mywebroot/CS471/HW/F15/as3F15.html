    Define a predicate grt_than/2 that takes two numerals in the notation that we introduced in this lecture (i.e. 0, sucC(0), sucC(sucC(0)) ...) as arguments and decides whether the first one is greater than the second one.
        e.g:

           ?- grt_than(sucC(sucC(sucC(0))),sucC(0)).

         sucC is an example of using nested terms. (This can be done with 2 clauses.)
