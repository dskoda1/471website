[Rosen 211#36] "Consider the following inductive definition of a version of Ackermann's function.  This function was named after Wilhelm Ackermann, a German mathematician who was a student of the great mathematician David Hilbert.  Ackermann's function plays an important role in the theory of recursive functions and in the study of complexity of certain algorithms involving set unions.  In the early days of  imperative languages, the Ackermann function was used to measure the recursion capability of a compiler." The Ackermann Number N of the compiler as the largest N for which the program computes the value.  Below is a simplication of the original function due to Rozsa Peter ( Other variations)
        5.  
        ack( m,n ) = 2n + 1
        		     if m = 0
        ack( m,n ) =  ack( m-1,1 )
        		     if m >=1 and n = 0 
        ack( m,n ) = ack( m-1, ack( m, n-1 ) )     if m >=1 and n >=1
