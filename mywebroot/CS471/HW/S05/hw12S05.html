1)
(From Abelson and Sussman [1985]) It is also possible to express the integer as lambda abstractions:
zero = l f  . l x . x
one =  lf.  l x ( f x)
two =  l f. l x ( f ( f x ))
. . . .

These are called Church numbers after Alonzo Church.

Given the following definition of the successor function:

successor = l n. l f. l x ( f (( n f ) x ) ).
    show that
    a) (successor zero) = one
    b) (successor one)  = two


2)
Given these definitions

init [x] = []
init (x:y:ys) = x : init (y:ys)
last [x] = x
last (x:y:ys) = last (y:ys)
[]++ ys = ys
(x:xs)++ ys = x : (xs ++ ys)

consider the proposition

init as ++ [last as] = as

  1. For what lists as is this proposition true?
  2. Prove that it holds for those lists.
