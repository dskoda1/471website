    2. [8pts]Prove that ack(m , 2) =  4.  (Prove by induction on m)
                  Base case: let m = 0
                                    ack(0,2) = 2*2 = 4
                    Assume: ack(m,2) = 4
                    Prove:  ack(m+1, 2)                             ?=  ack(m,2)
                                ack(m, ack(m+1, 1))                ?=  ack(m,2)
                                ack(m, 2)                                   =  ack(m,2)

                    QED

      Answer: The associative law of addition states that, for any real number x, y, and z, x + (y+z) = (x+y)+z. No matter how you parenthesize the addition of real numbers x1 + x2 + x3 + …. xn, the sum is the same as long as the numbers  appear in the same order.

      Answer: Subtraction on the set of integers is not commutative, e.g 4-6 != 6-4.

          Answer:  A predicate is a feature of language which you can use to make a statement about something, e.g. if you say "Peter
          is tall", then you have applied to Peter the predicate "is tall".
          Source: http://www.rbjones.com/rbjpub/logic/log019.htm

          Answer: Propositional logic is concerned with the logic of truth functional sentential or prepositional operators such as and, or
          and not. Predicate logic is prepositional logic with quantifiers all and some.
          Source: http://www.rbjones.com/rbjpub/logic/index.htm

  10. [10]There are two basic data structures used to build collections of items, arrays and linked lists. Each have advantages and disadvantages.
    1. What is the worst case time for accessing an element in an array?  In a link list?  Which data structure has the advantage for accessing elements?

      2. Answer:
      Array: O(1), since index is known
      Linked list: O(n), since needs to traverse from one node to the next node to access the element.
      Therefore, Array has advantage over linked list
    1. What is the worst case time for inserting/deleting  an element in an array? In a link list?  Which data structure  has the advantage for inserting/deleting elements?

      2. Answer:
      Inserting/Deleting Linear Array: O(n)
      Inserting/Deleting Circular Array (i.e. keep tracks of index that has current free space): O(1)
      Inserting/Deleting Linked List: O(1)
      Linked List is more advantageous. However, if specific element need to be identified before it's removed from the data strucutre, a complexity of searching O(n) will need to take into account.
       
    2. Describe how you would "grow" the size of an array.  What is the worst case time?  Describe how you would grow the size of link list.  What is the worst case time?  Which data structure has the advantage when you need to change the size?

      3. Answer:
      To grow the size of the array, a new array of size bigger than the old array need to be alloated. Then, the elements from the old arrays need to be copied to the new arrays. Thus, complexity is O(n)
      To grow the size of the linked list, complexity is O(1).
      Thus, linked list is more advantageous than array.
