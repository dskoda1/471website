      1. Build a DTD to parse an XML document. The XML document will contain information about high school course offerings.  The root tag will be CoursesOfferings which has zero or more Department children.  Department has the name of the department as text (PCDATA) and one or more Course children.  Note the grammar does not allow the following rule <!ELEMENT Department (#PCDATA,Course+)> therefore your solution may not match the specifications perfectly.   The element Course will have the  children elements in the following order , Name, Instructor, Enrollment.
